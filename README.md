# CARBOT
CARBOT (π-Conjugated Aromatic Rule-Based Organic Transformation) is a rule-based molecule generator for π-conjugated hydrocarbons.


## Requirement
- python: 3.12.12
- rdkit: 2025.09.2

*Other versions may work, but have not been tested.


## Installation
```sh
git clone https://github.com/sugakgroup/CARBOT.git
cd CARBOT
```
